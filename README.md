> A University Project for the course **Software Engineering Project (SEPR) on the TU Wien**.

Modulo is a modular planning app for everyday life, studies, and projects. It is a flexible, easy-to-use web application that seamlessly links tasks, notes, and appointments. The app helps young adults, students, and shared flats organize their daily lives together in a way that can be tailored to their individual needs.

---

## Features

* **Modular Boards:** The heart of the app. Boards serve as customizable containers that can be nested up to a costume depth.
* **Diverse Components:** Boards can be populated with a wide variety of content, including text, images, videos, tasks, notes, sketches, and calendars.
* **Collaboration & Group Management:** Users can create groups to collaborate on boards—ideal for shared flats or university projects. Detailed roles (**Group Owner, Writer, Reader**) can be assigned within groups.
* **Templates & Sharing:** Boards can be saved as templates and easily shared with other users or groups.
* **Calendar Integration:** External calendars can be imported via links or `.ics` files. Appointments can also be automatically converted into tasks.
* **Responsive UI:** The WebApp is fully responsive and optimized for mobile devices.

---

## 🛠️ Tech Stack

The project is based on a client-server architecture and uses RESTful communication via HTTP.

* **Frontend:** Angular
* **Backend:** Java Spring Boot
* **Database:** H2 Database (relational)
* **Build Tools & Package Managers:** Maven (Backend) and npm (Frontend)
